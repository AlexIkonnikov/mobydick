# Getting started

## Introduction

`@lad-tech/mobydick-markdown` is styled Markdown components based on [`@lad-tech/mobydick-core`](../core/Introduction) `react-native-markdown-display`

## Installation

```bash
npm install @lad-tech/mobydick-core react-native-markdown-display @lad-tech/mobydick-markdown
```
