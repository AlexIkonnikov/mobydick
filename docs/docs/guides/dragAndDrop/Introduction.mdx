# Getting started

## Introduction

`@lad-tech/mobydick-drag-and-drop` is styled Drag and Drop components based on [`@lad-tech/mobydick-core`](../core/Introduction), `react-native-reanimated` and `react-native-gesture-handler`

## Installation

```bash
npm install@lad-tech/mobydick-core react-native-reanimated react-native-gesture-handler @lad-tech/mobydick-drag-and-drop
```
