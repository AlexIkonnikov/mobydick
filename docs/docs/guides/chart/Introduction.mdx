# Getting started

## Introduction

`@lad-tech/mobydick-chart` is styled and highly performant charts, based on [`@lad-tech/mobydick-core`](../core/Introduction) [`react-native-reanimated`](https://docs.swmansion.com/react-native-reanimated/), [`react-native-gesture-handler`](https://docs.swmansion.com/react-native-gesture-handler/) and [`@shopify/react-native-skia`](https://shopify.github.io/react-native-skia/)

## Installation

```bash
npm install @lad-tech/mobydick-core @lad-tech/mobydick-chart @shopify/react-native-skia react-native-reanimated react-native-gesture-handler
```
