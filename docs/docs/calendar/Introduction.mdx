# Getting started

## Introduction

`@lad-tech/mobydick-calendar` is styled Calendar based on [`@lad-tech/mobydick-core`](../core/Introduction) and `react-native-calendars`

## Installation

```bash
npm install @lad-tech/mobydick-core @lad-tech/mobydick-calendar react-native-calendars
```
